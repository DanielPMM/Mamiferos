# 🐾 Sistema de Mamíferos / Mammal System - OOP in JavaScript (Node.js)

Este projeto foi desenvolvido para praticar os conceitos de Programação Orientada a Objetos (POO) utilizando JavaScript no ambiente Node.js.

This project was developed to practice Object-Oriented Programming (OOP) concepts using JavaScript in a Node.js environment.

---

## 🚀 Tecnologias / Technologies Used

* Node.js
* JavaScript (ES6+)
* Programação Orientada a Objetos (POO) / Object-Oriented Programming (OOP)
* Módulos ES (import/export) / ES Modules (import/export)

---

## 📌 Conceitos aplicados / Applied Concepts

* Classes e Objetos / Classes and Objects
* Herança (`extends`) / Inheritance (`extends`)
* Polimorfismo / Polymorphism (method overriding)
* Encapsulamento de atributos / Encapsulation of attributes
* Reutilização de código / Code reuse
